Description

Type A: is made of plain-woven polypropylene fabric and provides no static protection. The general bulk bag, jumbo bag, and container bag to transport the product...

Type B

Besides, Type B Jumbo bags are made from insulating fabric that has a breakthrough voltage of less than 6KV, this can prevent propagating brush discharges, but is not considered anti-static jumbo bags.

Type C (Conductive Jumbo Bag) is made from non-conductive PP fabric interwoven with conducting threads sewn in a grid pattern and connected to the ground via a grounding point...
